<br>

### 프론트엔드 애플리케이션 생성
1) 메인 폴더에서 바로 `npx create-react-app todo-fe` (대문자 사용불가)
2) React 버전 충동 발생
3) `cd todo-fe`
4) `npm install react@18 react-dom@18`
5) `npm i web-vitals`
6) `npm start` 정상적으로 실행되는지 확인
<br>

### SPA(Single Page Application)
